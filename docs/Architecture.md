# Architecture Design: GrowLedger

## System Overview
High-level architectural structure of GrowLedger-AI, illustrating how the React frontend interfaces with the Flask backend API.

## Component details
- **Frontend (React)**: High-performance, rich UI utilizing vanilla CSS and modern layout systems.
- **Backend (Python/Flask)**: Robust REST API handling business logic, ML inference, and explanation generation.
- **ML Engine**: Python-based pipeline for synthetic data generation, feature engineering, prediction, and SHAP/counterfactual explanation methods.
